# Spelling Game (เกมสะกดคำ) — Design

วันที่: 2026-06-28
สถานะ: อนุมัติดีไซน์แล้ว

## 1. ภาพรวม

เกมที่ 5 ของ kid-games: ฝึกสะกดคำภาษาอังกฤษทีละตัวอักษร สำหรับเด็ก 4-6 ขวบ
แสดงโจทย์ (ภาพ/เลข/สี) + แถวช่องสะกดที่มีตัวอักษรโชว์ไว้ 1 ตัว แล้วให้เด็กหยิบตัวอักษรจากแป้นไปเติมช่องให้ครบและถูกต้อง

## 2. หมวดเนื้อหา (3 หมวด)

1. **คำศัพท์รูปภาพ (vocab)** — โจทย์เป็นอิโมจิ เช่น 🐱 → สะกด `CAT` (คำ 3-5 ตัวอักษร)
2. **ตัวเลข (numbers)** — โจทย์เป็นตัวเลข เช่น `3` → สะกด `THREE`
3. **สี (colors)** — โจทย์เป็นช่องสี เช่นช่องแดง → สะกด `RED`

ผู้เล่นเลือกหมวดก่อนเริ่มเล่น เก็บดาวสูงสุดแยกรายหมวด (`spell:vocab`, `spell:numbers`, `spell:colors`)

## 3. กลไกการเล่น (ต่อ 1 คำ)

1. แสดงโจทย์ตามหมวด + แถวช่องสะกด โดยมีตัวอักษร **1 ตัว สุ่มตำแหน่ง** โชว์ไว้และ **ล็อก** (แก้ไม่ได้)
2. **แตะช่องว่าง** เพื่อเลือกช่องที่จะเติม (ไฮไลต์) ถ้าไม่ได้เลือก ระบบเติมช่องว่างซ้ายสุด
3. **แตะตัวอักษรในแป้น** → ตัวอักษรลงช่องที่เลือก และตัวนั้นหายจากแป้น (ใช้ซ้ำไม่ได้)
4. **แตะช่องที่เติมแล้ว** (ที่ไม่ล็อก) → คืนตัวอักษรกลับแป้น
5. เมื่อเติม **ครบทุกช่อง** → ตรวจอัตโนมัติ:
   - ถูกทุกตำแหน่ง → อ่านออกเสียงคำ + ฉลอง + ไปคำถัดไป (ได้แต้ม)
   - ผิด → สั่น + คืนตัวอักษรที่เติม (ที่ไม่ล็อก) กลับแป้น ให้ลองใหม่
6. นับว่าถูก (ได้แต้ม) **เฉพาะคำที่สะกดถูกตั้งแต่ตรวจครั้งแรก** ของคำนั้น

**รอบการเล่น:** 10 คำต่อรอบ — ดาวเกณฑ์เดิม (8-10=⭐⭐⭐, 5-7=⭐⭐, 1-4=⭐)

## 4. แป้นตัวอักษร

- รวมตัวอักษรที่ต้องใช้ทั้งหมดของคำ **ยกเว้นตัวที่โชว์ไว้** (รองรับตัวซ้ำ เช่น BALL ต้องมี L สองใบ)
- บวกตัวลวงสุ่ม **4 ตัว** ที่ไม่จำเป็นต้องใช้ (เลือกจาก A-Z) — ถ้าตัวลวงบังเอิญเป็นตัวเดียวกับที่ต้องใช้ ก็แค่เพิ่มจำนวนใบ ไม่เป็นไร
- สับลำดับทั้งหมด แต่ละใบมี id ไม่ซ้ำ (เป็น "ใบ" ของตัวอักษร ไม่ใช่ตัวอักษรเดี่ยว)

## 5. สถาปัตยกรรม

เกมนี้กลไกต่างจาก `QuizEngine` (เลือก 1 จาก N) จึงแยกเป็น engine ใหม่ แต่ใช้ส่วนกลางร่วมกัน:
`scoring` (ดาว), `audio` (พูด/เสียงตอบรับ), `storage` (ดาวสูงสุด), `Result` (หน้าสรุป)

### หน่วยงาน (units)

- `src/games/spelling/content.ts` — รายการคำ 3 หมวด: vocab (คำ+อิโมจิ), numbers (1-10), colors (ชื่อ+hex)
- `src/games/spelling/generate.ts` — `generateSpellQuestion(category)` คืน `SpellQuestion`
- `src/games/spelling/check.ts` — ฟังก์ชันบริสุทธิ์ `isComplete(slots)`, `isCorrect(slots, target)`
- `src/engine/SpellEngine.tsx` — คุมรอบ 10 คำ, state ของช่อง/แป้น, ตรวจ, นับดาว, เรียก `onFinish`/`onExit`
- `src/components/SlotRow.tsx` — แถวช่องสะกด (ช่องล็อก/ว่าง/เติมแล้ว/เลือกอยู่)
- `src/components/LetterBank.tsx` — แป้นตัวอักษร (ใบที่ยังไม่ถูกใช้)
- `src/components/CategoryPicker.tsx` — หน้าเลือกหมวด (3 ปุ่มใหญ่)
- แก้ `App.tsx` — เพิ่ม screen `category` และ `spell`
- แก้ `Home.tsx` — เพิ่มการ์ด "สะกดคำ" (แตะ → ไปหน้าเลือกหมวด)

### ชนิดข้อมูล

```ts
type SpellCategory = 'vocab' | 'numbers' | 'colors'

type SpellPrompt =
  | { kind: 'picture'; emoji: string }
  | { kind: 'number'; value: number }
  | { kind: 'color'; hex: string; name: string }

type LetterTile = { id: string; letter: string }

type SpellQuestion = {
  target: string            // คำคำตอบ ตัวพิมพ์ใหญ่ เช่น "FISH"
  prompt: SpellPrompt
  revealedIndex: number     // ตำแหน่งตัวอักษรที่โชว์ไว้ (0-based) ใน target
  bank: LetterTile[]        // ใบตัวอักษรในแป้น (สับแล้ว)
}

// สถานะช่องระหว่างเล่น: null = ว่าง, LetterTile = เติมแล้ว
// ช่องตำแหน่ง revealedIndex ถูกตั้งค่าและล็อกตั้งแต่ต้น
type SlotState = (LetterTile | null)[]
```

## 6. การทดสอบ (Vitest, logic ล้วน)

- `generate.test.ts` — รัน generator หลายรอบทุกหมวด: `target` ไม่ว่าง, `revealedIndex` อยู่ใน 0..len-1,
  prompt ตรงกับหมวด, จำนวนใบในแป้น = (จำนวนตัวที่ต้องเติม) + 4 ตัวลวง, แป้นมีตัวอักษรที่ต้องใช้ครบรวมตัวซ้ำ
  (เมื่อรวมกับตัวที่โชว์แล้วสะกดเป็น target ได้), ทุกใบมี id ไม่ซ้ำ
- `check.test.ts` — `isComplete` จริงเมื่อทุกช่องไม่ใช่ null; `isCorrect` จริงเฉพาะเมื่อตัวอักษรทุกช่องตรง target
  ตามตำแหน่ง (รวมกรณีตัวซ้ำ และกรณีเติมผิดช่อง)
- UI (SlotRow, LetterBank, SpellEngine, CategoryPicker) ตรวจด้วย build + preview

## 7. รายการคำเริ่มต้น

- **vocab** (อิโมจิ, คำ): 🐱 CAT, 🐶 DOG, 🐟 FISH, ☀️ SUN, ⭐ STAR, 🐦 BIRD, 🚗 CAR, 📖 BOOK,
  🌳 TREE, 🐸 FROG, 🐻 BEAR, 🦆 DUCK
- **numbers**: 1-10 → ONE..TEN (โจทย์แสดงตัวเลข)
- **colors**: RED, ORANGE, YELLOW, GREEN, BLUE, PURPLE, PINK, BROWN, BLACK, WHITE (โจทย์แสดงช่องสีจาก hex เดิม)

## 8. ตัดออกโดยตั้งใจ (YAGNI)

- ไม่มีระดับความยากปรับเอง (จำนวนตัวลวงคงที่ = 4)
- ไม่มีโหมดไม่โชว์ตัวอักษร (โชว์เสมอ 1 ตัว)
- ไม่บังคับลำดับการเติม (เติมช่องไหนก่อนก็ได้)
